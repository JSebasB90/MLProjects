Acá están varios proyectos trabajados:

## Proyecto Cartera

Para abrir este archivo

``` sh
git clone
cd CreditScore
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```

Para ejecutar el entorno Server visitar http://localhost:8080/predict después de

``` sh
python3 server.py
```
